# 材料数据质量评价系统

## 系统概述

本系统是一个基于人工智能的材料数据质量评价平台，专门用于评估**XRD（X射线衍射）**和**S-N（应力-寿命）疲劳试验**数据的质量。系统集成了大语言模型（LLM）的定性分析能力和ASTM E739标准的定量统计分析方法，为材料科学研究提供专业、准确、可定制的数据质量评估服务。

## 核心技术特性

- **AI驱动评价**: 基于大语言模型的智能数据质量评估，具备专业的材料科学知识
- **自定义提示词**: 可视化编辑和实时应用提示词模板，灵活适配不同评价标准
- **双重评价体系**: LLM定性分析与E739定量统计分析相结合，确保评价结果的全面性
- **多数据源支持**: 文件上传、数据库查询、API集成等多种数据输入方式
- **专业标准**: 严格基于ASTM E466/E739等国际标准
- **精准评分**: A/B/C/D四级评价体系，权重可配置，评分标准透明可控

## 详细功能说明

### XRD数据质量评估

#### 评估维度（5个核心维度）
| 维度 | 描述 | 权重 |
|------|------|------|
| 信息完整性 | 样品信息、仪器参数、测试条件的完整程度 | 可配置 |
| 数据规范性 | 数据格式、单位标注、坐标轴的规范程度 | 可配置 |
| 内容一致性 | 数据与描述的一致性、逻辑合理性 | 可配置 |
| 过程可追溯性 | 实验过程记录、参数设置的可追溯性 | 可配置 |
| 智能可用性 | 数据的机器可读性、后续分析适用性 | 可配置 |

#### 评价特点
- **可视化提示词编辑**: 右侧编辑框实时修改提示词
- **权重动态调整**: 滑块调整各维度权重，提示词自动更新
- **多格式支持**: 文本描述、PDF文件、数据库查询
- **等级划分**: A级(90-100) / B级(80-89) / C级(70-79) / D级(<70)

### S-N疲劳数据质量评估

#### 评估模式选择

**模式1: 纯LLM评估**
- 基于ASTM E466标准的专家级评价
- 适用于快速评估和定性分析
- 支持自定义提示词和评价标准

**模式2: LLM + E739统计分析**
- **LLM定性分析 (60%)**：基于ASTM E466标准的专业评价
- **E739定量分析 (40%)**：S-N曲线拟合、离群点检测、统计检验
- **综合评分**：两种方法结果的加权融合


#### E739统计分析功能
- **S-N曲线拟合**: 多种拟合模型（线性、幂函数、对数）
- **离群点检测**: 基于残差分析和影响点检测
- **统计检验**: 异方差检验、正态性检验
- **置信区间**: 预测区间和置信带计算
- **数据质量评分**: 基于统计指标的量化评分

#### 核心文件功能说明

##### 配置文件 (config/)
- **xrd_config.yaml**: XRD评价的完整配置
  - 评价维度定义（信息完整性、数据规范性等）
  - 权重配置（各维度重要性分配）
  - 提示词模板（FILE_EXTRACTION、SCORING_TEMPLATE）
  - 等级阈值（A/B/C/D级别划分标准）

- **sn_config.yaml**: S-N疲劳数据评价配置
  - ASTM E466/E739标准集成
  - 评价维度（试验条件、试样信息、数据质量等）
  - E739统计分析权重（LLM 60% + E739 40%）
  - 提示词模板（DEFAULT_ANALYSIS等）

- **database_config.yaml**: 数据库连接配置
  - 支持MySQL、MongoDB、PostgreSQL
  - 连接池配置和认证信息
  - 环境变量和加密存储支持

##### 通用模块 (src/common/)
- **llm_client.py**: 统一的LLM API客户端
  - 支持多种AI模型（OpenAI、Claude、本地模型）
  - 自动重试和错误处理
  - 请求日志和响应格式化

- **data_processor.py**: 数据处理核心引擎
  - 多格式文件解析（PDF、CSV、Excel、JSON）
  - 数据验证和清洗
  - XRD/S-N数据格式标准化

- **data_sources.py**: 数据源抽象层
  - 统一的数据源接口
  - 工厂模式创建数据源
  - 支持文件、数据库、API等多种来源

##### 分析模块
- **XRD分析器**: 基于文本描述的质量评价
  - 支持PDF文件信息自动提取
  - 动态提示词生成和评分计算
  - 数据库集成查询和批量处理

- **S-N分析器**: 疲劳数据专业评价
  - ASTM E466标准符合性检查
  - E739统计分析集成（S-N曲线拟合、离群点检测）
  - 定性分析与定量分析结果融合

### 提示词工作流程

#### 数据处理流程
```
数据输入 → 提示词处理 → AI分析 → 结果输出
    ↓           ↓           ↓         ↓
用户输入    提示词选择    LLM调用    格式化输出
│           │           │          │
├─文本描述   ├─文件提取    ├─质量评估   ├─评分报告
├─文件上传   ├─评分模板    ├─统计分析   ├─改进建议
└─数据库     └─自定义     └─综合判断   └─可视化图表
```

#### 提示词管理机制

**系统启动流程**
```
配置文件加载 → 提示词模板解析 → 界面初始化 → 用户交互就绪
```

**实时编辑流程**
```
界面编辑 → 语法验证 → 点击应用 → 内部状态更新 → 后续评估生效
```

**参数联动机制（XRD专用）**
```
权重滑块调整 → 提示词自动重新生成 → 界面同步更新 → 状态一致性保持
```
